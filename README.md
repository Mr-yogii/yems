# Y-EMS (Yogii Expense Management System)

A lightweight, production-ready **Expense Management System** built by **Yogeshwaran (Yogii)** using **HTML5, CSS3, Vanilla JavaScript, Google Apps Script, Google Sheets, and Google Drive**.

## 🚀 Overview

Y-EMS helps organizations manage daily expenses through a simple web interface while storing data securely in Google Sheets. Receipts are uploaded to Google Drive, making the system completely serverless and easy to deploy.

## ✨ Features

- Secure login authentication
- Dashboard with expense summary
- Add, edit, and delete expenses
- Receipt/document upload to Google Drive
- Reports with export support (CSV, Excel, PDF & Print)
- Charts and analytics
- Google Sheets as the database
- Responsive UI for desktop and mobile

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Google Apps Script
- Google Sheets
- Google Drive
- Chart.js

## ⚡ Deployment

1. Deploy the Google Apps Script as a **Web App**.
2. Copy the generated Web App URL.
3. Update `WEB_APP_URL` in `js/config.js`.
4. Push the project to GitHub.
5. Enable **GitHub Pages** to host the frontend.
